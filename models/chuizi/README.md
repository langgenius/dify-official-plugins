# Chuizi.AI

Chuizi.AI is a unified AI gateway providing access to 100+ models across 16 providers through a single OpenAI-compatible endpoint.

## Configuration

1. Get your API key at [app.chuizi.ai](https://app.chuizi.ai/login)
2. In Dify, go to Settings > Model Provider
3. Add Chuizi.AI and enter your API key

## Predefined Models

- Claude Sonnet 4.6, Opus 4.6, Haiku 4.5
- GPT-4.1, GPT-4.1 Mini, o4-mini
- Gemini 2.5 Pro
- DeepSeek V3.2, DeepSeek R1

## Custom Models

Use the `provider/model` naming format to access any model available on Chuizi.AI. For example:
- `qwen/qwen3-max`
- `mistral/mistral-large-3`
- `meta/llama-3.2-90b`

See the full model list at [chuizi.ai/models](https://chuizi.ai/models).
