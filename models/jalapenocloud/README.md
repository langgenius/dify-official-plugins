## Overview

JalapenoCloud is an enterprise-grade AI compute platform that provides access to
various large language models through a unified OpenAI-compatible API.

## Configure

Install the JalapenoCloud plugin, then configure it by entering your API Key.
Get your API Key from [JalapenoCloud](https://www.jalapeno-cloud.ai/).

## About JalapenoCloud

JalapenoCloud offers ultra-low latency inference, industrial-grade throughput,
and a unified OpenAI-compatible API for instant integration.

- Website: https://www.jalapeno-cloud.ai/
- API Base URL: https://api.jalapeno-cloud.ai/v1
