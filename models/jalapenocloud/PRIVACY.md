# Privacy Policy

## Data Collection

This plugin collects the following data when in use:

- **API Key**: Your JalapenoCloud API key, used to authenticate requests to the
  JalapenoCloud API.

## Data Usage

- The API key is used solely to authenticate requests between the Dify instance
  and JalapenoCloud's API.
- No other user data is collected or stored by this plugin.
- Prompt content and model responses are transmitted to JalapenoCloud's API for
  processing, subject to JalapenoCloud's privacy policy.

## Data Storage

- The API key is stored encrypted within the Dify platform.
- No data is stored locally by the plugin itself.

## Contact

For privacy inquiries, contact: support@jalapeno-cloud.ai
