# Privacy Policy

This plugin connects to the IO Intelligence API provided by io.net.

## Data Collection

This plugin sends the following data to the IO Intelligence API:
- Your API key for authentication
- Prompt messages and model parameters for inference requests

## Data Storage

This plugin does not store any user data locally. All inference requests are processed by IO Intelligence servers.

## Third-Party Services

This plugin communicates with:
- `api.intelligence.io.solutions` - IO Intelligence API endpoint

For more information about IO Intelligence's privacy practices, please refer to [io.net's privacy policy](https://io.net/privacy).
