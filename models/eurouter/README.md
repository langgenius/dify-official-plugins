# EUrouter Plugin for Dify

EUrouter is a European AI Gateway providing a single OpenAI-compatible API endpoint to EU-hosted, GDPR-friendly AI models.

## Setup

1. Get an API key from [EUrouter](https://www.eurouter.ai)
2. Add EUrouter as a model provider in Dify
3. Enter your EUrouter API key
4. Select from available models

## Features

- All models hosted in the European Union
- GDPR compliant by design
- Smart routing and safe fallbacks
- OpenAI-compatible API
- Streaming and function calling support
- Both predefined and customizable models
