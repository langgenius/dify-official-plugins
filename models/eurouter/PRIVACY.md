## Privacy

This plugin does not collect, store, or process any personal data on its own. All data is sent directly to the EUrouter API endpoint (https://api.eurouter.ai/api/v1) for processing.

EUrouter routes all requests to EU-hosted inference providers. No data leaves the European Union. All processing is GDPR compliant by design.

For more information about data handling, please refer to:
- EUrouter Privacy Policy: https://www.eurouter.ai/privacy
- EUrouter Website: https://www.eurouter.ai
