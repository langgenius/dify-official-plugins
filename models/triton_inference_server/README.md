# Triton Inference Server

Triton Inference Server lets Dify connect to NVIDIA Triton-hosted model inference endpoints.

## Features
- Provides llm models in Dify.
- Supports customizable model configuration.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Get the required credentials from [Triton Inference Server](https://github.com/triton-inference-server/server).
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Select **Triton Inference Server** as the model provider in Dify, choose an available model, and use it in applications, agents, or workflows.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
