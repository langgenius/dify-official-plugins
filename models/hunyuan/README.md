# Hunyuan

Models provided by Tencent Hunyuan, such as hunyuan-standard, hunyuan-standard-256k, hunyuan-pro, hunyuan-role, hunyuan-large, hunyuan-large-role, hunyuan-turbo-latest, hunyuan-large-longcontext, hunyuan-turbo, hunyuan-vision, hunyuan-turbo-vision, hunyuan-functioncall and hunyuan-lite.

## Features
- Provides llm, text-embedding models in Dify.
- Includes predefined llm models such as hunyuan-functioncall, hunyuan-large-role, hunyuan-large-longcontext.
- Includes predefined text embedding models such as hunyuan-text-embedding.
- Supports predefined model configuration.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Get the required credentials from [Hunyuan](https://console.cloud.tencent.com/cam/capi).
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Select **Hunyuan** as the model provider in Dify, choose an available model, and use it in applications, agents, or workflows.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
