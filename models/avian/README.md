## Overview

Avian provides an OpenAI-compatible API for accessing a curated set of high-performance LLMs at competitive pricing. Available models include DeepSeek V3.2, Kimi K2.5, GLM-5, and MiniMax M2.5, with support for chat completions, streaming, and function calling.

## Configure

To use Avian, you'll need an API key from [avian.io](https://avian.io). After installing the plugin, find the Model Provider settings and enter your API key. Save to activate.
