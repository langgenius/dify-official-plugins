# 360 AI

Models provided by 360 AI.

## Features
- Provides llm models in Dify.
- Includes predefined llm models such as 360gpt-turbo, 360gpt-turbo-responsibility-8k, 360gpt2-pro.
- Supports predefined model configuration.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Get the required credentials from [360 AI](https://ai.360.com/platform/keys).
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Select **360 AI** as the model provider in Dify, choose an available model, and use it in applications, agents, or workflows.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
