# 01.AI

Models provided by 01.AI, such as yi-34b-chat and yi-vl-plus.

## Features
- Provides llm models in Dify.
- Includes predefined llm models such as yi-34b-chat-200k, yi-vl-plus, yi-large-turbo.
- Supports predefined model and customizable model configuration.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Get the required credentials from [01.AI](https://platform.lingyiwanwu.com/apikeys).
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Select **01.AI** as the model provider in Dify, choose an available model, and use it in applications, agents, or workflows.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
