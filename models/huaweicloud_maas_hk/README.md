# Huawei Cloud MaaS - HK

Huawei Cloud MaaS provides access to various models (LLMs), configurable via model name, API key, and other parameters.

## Features
- Provides llm models in Dify.
- Includes predefined llm models such as deepseek-r1, deepseek-v3.2, deepseek-v3.1.
- Supports predefined model and customizable model configuration.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Get the required credentials from [Huawei Cloud MaaS - HK](https://console-intl.huaweicloud.com/modelarts/?locale=zh-cn&agencyId=3ff2dd2fd7fc4ba7a0c1ff02564fba83®ion=ap-southeast-1#/model-studio/authmanage).
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Select **Huawei Cloud MaaS - HK** as the model provider in Dify, choose an available model, and use it in applications, agents, or workflows.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
