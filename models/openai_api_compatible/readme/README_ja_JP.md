## 概要

このプラグインは、LLM、リランキング、テキスト埋め込み、音声テキスト変換（STT）、テキスト音声変換（TTS）モデルを含む、OpenAI互換モデルへのアクセスを提供します。開発者は、モデル名やAPIキーなどの設定パラメータを提供することで、簡単にモデルを追加できます。

## 設定

コアの詳細（タイプ、名前、APIキー、URL）を提供し、補完、コンテキスト、トークン制限、ストリーミングおよびビジョン設定などのさらなるオプションを調整することで、OpenAI-API互換モデルを設定します。完了したら保存します。
