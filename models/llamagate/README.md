# LlamaGate

LlamaGate is an OpenAI-compatible API gateway for open-source LLMs with credit-based billing.

## Features

- **26+ Open-Source Models**: Access Llama, Mistral, DeepSeek R1, Qwen, and more
- **OpenAI-Compatible API**: Drop-in replacement for OpenAI SDK
- **Competitive Pricing**: $0.02-$0.55 per 1M tokens
- **Vision Models**: Qwen VL for multimodal tasks
- **Reasoning Models**: DeepSeek R1 for complex problem-solving
- **Code Models**: CodeLlama, DeepSeek Coder, Qwen Coder
- **Embedding Models**: Nomic Embed Text, Qwen3 Embedding

## Supported Models

### LLM Models
- Llama 3.1 8B Instruct
- Llama 3.2 3B
- DeepSeek R1 8B
- DeepSeek R1 Distill Qwen 7B
- Qwen 3 8B
- Mistral 7B Instruct v0.3
- Qwen 2.5 Coder 7B
- Qwen 3 VL 8B (Vision)
- OpenThinker 7B
- Dolphin 3 8B
- CodeLlama 7B
- DeepSeek Coder 6.7B

### Embedding Models
- Nomic Embed Text
- Qwen3 Embedding 8B

## Getting Started

1. Sign up at [llamagate.dev](https://llamagate.dev)
2. Get your API key from the dashboard
3. Configure the LlamaGate plugin in Dify with your API key
