# 关于胜算云
胜算云是由上海胜算速惠云科技打造的专业AI算力平台，旨在为用户提供经济实惠的GPU算力资源，为AI开发者 和厂商打造集开发与弹性推理为一体的云平台。拥有海量的GPU算力资源，让AI开发变得更加轻松。

# Configuration
安装成功后, 在 [胜算云](https://console.shengsuanyun.com/user/keys) 官网 个人中心 -> API 密钥页面生成和管理密钥。
