# Feishu Wiki

Feishu Wiki, requires the following permissions: wiki:wiki:readonly.

## Features
- Get Wiki Nodes: Get the list of child nodes in Wiki, make sure the app/bot is a member of the wiki space. See How to add an app as a wiki base administrator (member). https://open.feishu.cn/document/server-docs/docs/wiki-v2/wiki-qa.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Feishu Wiki tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
