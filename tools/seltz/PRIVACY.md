# Privacy Policy for Seltz Plugin

## Data Processing

This plugin sends search queries to the Seltz API (`api.seltz.ai`) to retrieve relevant documents and content.

## Data Collection

- **Search queries**: Your search text is sent to Seltz servers for processing
- **API key**: Stored securely in your Dify environment, sent to Seltz for authentication
- **Results**: Search results are returned from Seltz and displayed in Dify

## Data Transmission

- All data is transmitted over HTTPS (encrypted)
- Queries are processed by Seltz's AI-powered search infrastructure
- Please refer to Seltz's privacy policy for details on their data handling practices

## Data Storage

- This plugin does not store any user data locally
- Search history is not retained by the plugin
- API credentials are managed by Dify's secure credential storage

## Third-Party Services

This plugin relies on the Seltz API service. By using this plugin, you agree to Seltz's terms of service and privacy policy.

## Contact

For privacy-related questions about this plugin, please open an issue on the [dify-official-plugins repository](https://github.com/langgenius/dify-official-plugins).

For questions about Seltz's data practices, please contact Seltz directly at https://console.seltz.ai.
