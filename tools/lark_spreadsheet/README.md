# Lark Spreadsheet

Lark Spreadsheet, requires the following permissions: sheets:spreadsheet.

## Features
- Create Spreadsheet.
- Add Rows.
- Read Rows.
- Read Table.
- Get Spreadsheet.
- Read Cols.
- Add Cols.
- List Spreadsheet Sheets.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Lark Spreadsheet tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
