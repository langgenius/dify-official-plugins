# Feishu Task

Feishu Task, requires the following permissions: task:task:write、contact:user.id:readonly.

## Features
- Delete Task.
- Create Task: Create Feishu Task.
- Update Task: Update Feishu Task.
- Add Members.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Feishu Task tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
