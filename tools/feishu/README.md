# Feishu

Feishu group bot.

## Features
- Send Group Message: Sending a group message on Feishu via the webhook of group bot.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Open the plugin settings in Dify.
3. Save the configuration.

## Usage
Add the Feishu tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
