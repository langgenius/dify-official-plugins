# Lark Cloud Document

Lark cloud document, requires the following permissions: docx:document、drive:drive、docs:document.content:read.

## Features
- Create Lark document.
- Get Lark Cloud Document Content.
- Write Document: Adding new content to Lark documents.
- List Lark Document Blocks.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Lark Cloud Document tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
