# Front

Connect to Front API to send emails and manage conversations.

## Features
- Send Email: Send emails through Front API using available email channels.
- Find Conversations: Search and filter conversations in Front with various criteria like status, inbox, tags, and date range.
- Find Channels & Conversations: Search and filter Front channels and conversations with comprehensive filtering options.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Open the plugin settings in Dify.
3. Save the configuration.

## Usage
Add the Front tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
