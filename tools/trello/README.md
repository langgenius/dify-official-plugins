# Trello

Trello: A visual tool for organizing your work and life.

## Features
- Update Card by ID: Updates specified attributes of a Trello card, such as its name, description, list ID, and board ID, by providing the card's unique ID.
- Get Board by ID: Retrieves detailed information about a specific Trello board using its unique ID. This tool enables users to quickly access board details without navigating through the Trello interface.
- Get Filtered Board Cards: Retrieves cards from a Trello board using a specified filter and the board's ID. Filters include options like 'all', 'open', 'closed', 'none', and 'visible', allowing for tailored views of board content.
- Create List on Board: Creates a new list on a specified Trello board by providing the board's ID and the desired name for the list. Streamlines the process of organizing board content.
- Create New Card on Board: Creates a new card on a Trello board with specified details like name, description, list ID, and other optional parameters. Facilitates task addition and project management within Trello.
- Get Board Cards: Retrieves all cards present on a specific Trello board by its ID, providing a list of card names and their IDs. Useful for managing and organizing project tasks.
- Update Board by ID: Updates a Trello board's settings based on the provided ID and parameters. Allows for changing the board's name, description, status, and other preferences.
- Get Lists from Board: Retrieves all lists from a specified Trello board by its ID, providing an overview of the board's organization and current phases or categories.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: Trello API key, Trello API token.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Trello tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
