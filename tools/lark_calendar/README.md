# Lark Calendar

Lark calendar, requires the following permissions: calendar:calendar:read、calendar:calendar、contact:user.id:readonly.

## Features
- Add Event Attendees.
- Search Events.
- Get Primary Calendar.
- Create Event.
- Delete Event.
- List Events.
- Update Event.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Lark Calendar tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
