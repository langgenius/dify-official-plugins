# Dify Fish Audio プラグイン

## 概要

**Fish Audio** は、Fish Audio API を搭載した高度なテキスト読み上げ（TTS）ツールです。テキストを高品質な音声に変換し、さまざまな使用ケースに対応したカスタマイズ可能な音声オプションを提供します。仮想アシスタントの構築、オーディオブックの作成、ナレーションの生成など、Fish Audio は信頼性が高く効率的な TTS 機能を提供し、アプリケーションの機能を強化します。

## セットアップ

Fish Audio を使い始めるには、以下の手順に従ってください：

1. **Fish Audio ツールのインストール**
プラグインマーケットプレイスにアクセスし、Fish Audio ツールをインストールして、その TTS 機能をアプリケーションに統合します。

2. **Fish Audio API Key の申請**
[Fish Audio API Keys](https://fish.audio/zh-CN/go-api/api-keys) ページにアクセスして API Key を生成します。アカウントが TTS サービスにアクセスするために必要な権限を持っていることを確認してください。

3. **Fish Audio の認証**
**ツール > Fish Audio > 認証へ** に移動し、API Key を入力してツールを有効化します。

![](./_assets/fishaudio-01.png)

## ツール機能

Fish Audio ツールは、主に **TTS（テキスト読み上げ）** という 1 つのアクションを提供します。

### TTS

Fish Audio API を使用してテキストを音声に変換します。TTS アクションを使用するには、以下のパラメータを提供する必要があります：

* **Text**：音声に変換したいテキストコンテンツ。
* **Voice ID**：音声生成に使用する一意の音声識別子。Fish Audio は複数の音声オプションをサポートしており、カスタマイズが可能です。

## 使用方法

Fish Audio は **チャットフロー / ワークフローアプリ** および **エージェントアプリ** にシームレスに統合できます。

### チャットフロー / ワークフローアプリ

1. Fish Audio TTS ノードをチャットフローまたはワークフローパイプラインに追加します。
2. テキストコンテンツを提供し、Voice ID を選択してノードを設定します。
3. パイプラインが実行されると、TTS アクションが音声オーディオを生成し、再生したり、ワークフロー出力の一部として使用したりできます。

![](./_assets/fishaudio-03.png)

### エージェントアプリ

1. Fish Audio TTS ツールをエージェントアプリケーションに追加します。
2. インタラクション中に、チャットインターフェイスを通じてテキストコンテンツを送信します。
3. TTS ツールがテキストを音声に変換し、生成されたオーディオをユーザーに返します。

![](./_assets/fishaudio-04.png)
