# Feishu Base

Feishu base, requires the following permissions: bitable:app.

## Features
- Delete Tables: Batch Delete Data Tables from Multidimensional Table.
- List Tables: Get All Data Tables under Multidimensional Table.
- Create Base: Create Multidimensional Table in Specified Directory.
- Add Records: Add Multiple Records to Multidimensional Table.
- Search Records: Query records in a multidimensional table, up to 500 rows per query.
- Delete Records: Delete Multiple Records from Multidimensional Table.
- Create Table: Add a Data Table to Multidimensional Table.
- Read Records: Batch Retrieve Records from Multidimensional Table.

## Setup
1. Install this plugin from the Dify Marketplace.
2. Prepare the required credentials: APP ID, APP Secret.
3. Add the credentials in the plugin settings.
4. Save the configuration.

## Usage
Add the Feishu Base tools to an agent or workflow, fill in the required inputs, and run the node to call the upstream service.

## Privacy
This plugin sends the inputs required by the selected operation to the upstream service. Review the upstream service's privacy policy before use.
