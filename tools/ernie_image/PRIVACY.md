# ERNIE Image Privacy Policy

This plugin sends the prompt and generation parameters you provide to Baidu AI
Studio (`https://aistudio.baidu.com`) using the access token configured in the
plugin credentials. The prompt, the returned image URL, and the revised prompt
returned by the model are processed by Baidu under their own privacy terms; see
<https://aistudio.baidu.com>.

The plugin itself does not store, log, or share your prompts, credentials, or
generated images. Credentials live only in the Dify plugin settings and are
sent solely to Baidu's API as a Bearer token.
