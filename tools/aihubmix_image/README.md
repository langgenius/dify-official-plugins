# AIHubMix Image Generation Plugin

**Author:** AIHubMix

**Version:** 0.1.4

**Type:** Dify Plugin

## Overview

The AIHubMix Image Generation Plugin provides access to a variety of advanced image generation and image editing models through a unified AIHubMix API. It supports synchronous and asynchronous workflows, multiple resolutions, batch generation, image-to-image processing, and multilingual prompts.


## Supported Models

### Image Generation Models

* **GPT Image** (`gpt-image-2`, `gpt-image-1.5`, `dall-e-3`)
* **Microsoft MAI Image** (`mai-image-2.5`, `mai-image-2.5-flash`)
* **Gemini Nano Banana** (`gemini-3-pro-image`, `gemini-3.1-flash-image`, `gemini-3.1-flash-lite-image`, plus preview IDs)
* **Google Imagen** (Imagen 4.0 series)
* **Flux** (`FLUX.1-Kontext-pro`, `FLUX-1.1-pro`)
* **Qwen Image** (`qwen-image`, `qwen-image-2.0`, `qwen-image-2.0-pro`, Max, and Plus)
* **Wan** (`wan2.7-image`, `wan2.7-image-pro`)
* **Doubao Seedream** (`doubao-seedream-5.0-pro`, `doubao-seedream-5.0-lite`, `doubao-seedream-4-5`)
* **GLM Image**, **Ideogram V3**, and **ERNIE iRAG**

### Image Editing Models

* **GPT Image Edit** (`gpt-image-1.5`)
* **Qwen Image Edit** (`qwen-image-edit`, Plus, and Max)
* **ERNIE iRAG Edit** (`ernie-irag-edit`)
